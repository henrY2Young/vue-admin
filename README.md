# Features

其实项目很简单，主要的特点在于权限的控制，由于是前后端分离的项目，用户的权限划分是一个难题，永不不要相信用户的输入，所以在每一次的用户进入路由的时候配合后端进行判断，如果用户权限路由不存在则跳转到错误页面
利用的是JWT，所以如果想让项目运行起来需要一定的后端知识，同时需要下载后端的项目，后端是python+flask。
